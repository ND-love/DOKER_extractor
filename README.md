# Docker Extractor & Analyzer

Консольная утилита на Python для статического анализа образов Docker. Скрипт распаковывает `.tar` архивы (созданные командой `docker save`), самостоятельно производит слияние слоев и ищет внутри исполняемые файлы. 

Для всех найденных ELF-файлов и файлов с флагом исполнения (`+x`) утилита вычисляет MD5-хеш.

## Требования
* Python 3.6+

## Использование
Справка по командам:
```bash
python extractor.py --help
```
Перед запуском скрипта необходимо выгрузить нужный Docker-образ в формате `.tar`. Для этого используйте стандартную команду:
```bash
docker save -o image.tar <имя_вашего_образа>
```
Базовый запуск (сохранит результаты в `hashes_output.txt` в текущей директории):
```bash
python extractor.py image.tar
```
Указание собственного имени файла для отчета с помощью флага -o или --output:

```Bash
python extractor.py image.tar -o <имя_выходного_файла>.txt
```
Выходные данные
В результирующем текстовом файле генерируется список в формате:
[Теги] MD5-хеш /путь/внутри/контейнера

Пример:
```Plaintext
[ ELF,+x] 1234567890abcdef1234567890abcdef  /usr/bin/curl
[  +x   ] fedcba0987654321fedcba0987654321  /app/test_script.sh
```
